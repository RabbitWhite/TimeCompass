missions/
This folder contains mission briefs for Claude Code sessions.
next_mission.md — the current mission to execute. Read this at the start of every session.
completed/ — archive of completed missions, named YYYY-MM-DD-mission-name.md
When you open a Claude Code session on this repo, your first action is always to check whether missions/next_mission.md exists. If it does, read it and execute it. If it does not, wait for instructions.
